# Conveyare

![](conveyare.jpg)

Conveyare is a light routing library for [Kafka](https://kafka.apache.org) based micro-services.

## Installation

Add the following dependency to your `project.clj` file:

```clj
[com.ververve/conveyare "0.1.0"]
```
